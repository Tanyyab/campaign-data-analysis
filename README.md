# Campaign & Customer Behaviour Analysis — SQL

## Overview
SQL-based analysis across 50+ campaigns and customer behaviour 
datasets to identify drop-off patterns, data inconsistencies, 
and pipeline performance gaps.

## Business Problem
Conversion data showed 15–25% discrepancies across campaign 
reporting — impacting budget allocation decisions.

## What I Did
- Queried raw campaign data, customer interaction logs, 
  and conversion records
- Identified root causes of data discrepancies
- Surfaced drop-off patterns in customer behaviour

## Results
- Reduced data discrepancy rate from ~20% to less than 5%
- Cut resolution cycle time by 30%
- Standardised data validation workflows across 50+ campaigns

## Tools Used
SQL | Power BI | Google Analytics | Excel
